Age Detection AI Web App

An AI-powered web application that detects human faces in images and predicts their age group using Deep Learning models. Built with OpenCV and deployed using Streamlit, this project provides a simple and interactive interface for real-time age detection.
Features =>
📤 Upload image and detect faces instantly
🧠 Deep Learning-based age prediction
🎯 Accurate face detection using OpenCV DNN
🌐 Interactive web app using Streamlit
⚡ Fast and lightweight performance

Tech Stack =>
Python 
OpenCV (DNN Module)
NumPy
Streamlit
Deep Learning (Caffe Models)

Project Structure =>
├── app.py
├── opencv_face_detector.pbtxt
├── opencv_face_detector_uint8.pb
├── age_deploy.prototxt
├── age_net.caffemodel
├── requirements.txt
└── README.md
